hexo 目录如下：

    .
    ├── _config.yml 
    |
    ├── package.json 
    |
    ├── scaffolds 
    |
    ├── scripts 
    |
    ├── source 
    |
    |   ├── _drafts 
    |   |
    |   └── _posts 
    |
    └── themes 

参考：
https://hexo.io/zh-cn/docs/

新建文章：
    ```hexo new title```
生成：
    ```hexo generate```
服务器：
    ```hexo server```
部署：
    ```hexo deploy```
